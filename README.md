<h2>
  💡 How to run the project?
</h2>

### Clone the repository to have a copy of the code on your machine

```bash
$ git clone https://github.com/DeboraZandonai/Bowe.git && Bowe
```

### Load dependencies

```bash
$ yarn
```

#### or

```bash
$ npm install
```

# Run the project

```bash
$ yarn start
```

#### or

```bash
$ npm start
```

## Routes

```bash
$ /contact
```
